О моей работе

Мои основные направления — веб-разработка и создание ботов для Telegram. Я создаю современные веб-приложения с использованием актуальных технологий и разрабатываю функциональные Telegram-боты для автоматизации бизнес-процессов.

Мой подход сочетает техническое мастерство с пониманием потребностей клиентов, чтобы создавать не только функциональные, но и удобные решения.

Навыки

• Веб-разработка (HTML, CSS, JavaScript, React, Node.js)

• Создание Telegram-ботов (Python, Node.js, aiogram, node-telegram-bot-api)

• Backend разработка (Node.js, Python, Express)

• Работа с API и базами данных

• Интеграция ботов с веб-сервисами
